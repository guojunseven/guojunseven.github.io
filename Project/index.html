My Projects

A scalable distributed
text analysis web application

Description: I Implemented the web server using Servlet and deployed it through Tomcat on 4 * t2.micro AWS elastic computing servers with a network level elastics load balancer. I Improved the throughput of concurrent requests up to 8000 / s through implementing asynchronous message queues with RabbitMQ with custom channel pool and using the efficient DynamoDB cluster. I Designed the Backend using serverless processing stream and CQRS pattern which applied AWS lambda streaming service listening to the update events for the dynamo database and processing the text to Aurora database for efficient query.

A full cycle
code risk evalution service

Description: Developed a service called Quality Metrics Unit using Python in the CI phase of Integrated Pipeline with DevOps fashion which utilized Logistic Regression for evaluating the code risk based on code features and expertise knowledge for saving 90% manual tests. I was responsible for the entire life cycle of the service including training model, developing RESTful API and evaluation web page for labeling and reviewing.
